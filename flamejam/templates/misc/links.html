Resources

Here are some game development resources for you. Always make sure to check the licence!

  • Google docs link - a google docs document which is provided by the GlobalGameJam community with a big lis of resources.
  • OpenGameArt.org - open collection of artwork for games. You can find sprites, models and sounds here.
  • CgTextures.com - huge collection of freely available high quality textures. Download limit for free accounts.
  • as3sfxr and bfxr - two sound effect synthesizers for retro style games.
  • The Video Game Name Generator - in case you're stuck for ideas.
  • ludumdare tools list - a list of tools for game development.
If you have something to add, please tell us by send a quick email.

Other game jams

  • GlobalGameJam.org - once each year the whole world is jamming.
  • LudumDare.com - probably one of the bigger gamedev communities, separated into a competition and a jam.
  • PyWeek.org - one-week python game jam, usually twice per annum.
  • BaconGameJam.org - mmmhhh bacon
